# Greenova Life

Official website of **Greenova Life**, a non-profit organization dedicated to sustainability, environmental awareness, community development, and innovative green initiatives.

---

## Technology Stack

- React
- TypeScript
- Vite
- Tailwind CSS

---

## Development

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
